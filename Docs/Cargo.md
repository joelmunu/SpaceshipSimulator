# Clase Cargo
## Atributos
En la clase cargo se encuntran los siguientes atributos:
* cargaMaxima
* cargaActual
* contenedoresACagar

## Métodos
### Método load()
> Permite cargar contenedores en la nave, hasta que se llene.

### Método unload()
> Permite al usuario descargar los contenedores cargados, hasta vaciar la nave.
